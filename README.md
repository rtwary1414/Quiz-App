# Quiz App (SIT708 Credit Task 3.1)

## Overview
This is an Android Quiz Application developed in Java. The app allows users to attempt multiple-choice questions, track their progress in real 
time, and view their final score. It also supports Light and Dark mode with persistent theme settings.

## Features

- Multiple-choice quiz with 5 questions  
- Real-time progress tracking using ProgressBar  
- Answer validation with visual feedback  
  - Correct answer → Green  
  - Incorrect answer → Red  
- Prevents answer changes after submission  
- Final score display with user name  
- Restart quiz functionality  
- App exit functionality  
- Light Mode / Dark Mode toggle  
- Theme persistence using SharedPreferences  

## Technologies Used

- Java (Android Development)
- Android Studio
- XML (UI Design)
- SharedPreferences (Data Persistence)
